This is a plain Java maven project for Trading Transaction processing

*Instructions
1.The trading data input is in TradeData.json file
2.The data for trade for a particular date is in SettelementData.json file for the prices of an entity on that day
3. the selling buying criteria is not matched then the trade goes unsetteled 
4. The results are been displayed in the console and even written to the files in D drives
	IncomingSettelements.txt
	OutGoingSettelements.txt
	IncomingRankSettelements.txt
	OutgoingRankSettelements.txt
